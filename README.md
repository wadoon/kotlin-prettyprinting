# kotlin-prettyprint
![Maven Central Version](https://img.shields.io/maven-central/v/io.github.wadoon/kotlin-prettyprinting)

Pretty printing library for Kotlin. A port of the [pprint in ocaml](https://github.com/fpottier/pprint) to the JVM.

[Maven coordinates](https://central.sonatype.com/artifact/io.github.wadoon/kotlin-prettyprinting):

```gradle 
implementation("io.github.wadoon:kotlin-prettyprinting:$version")
```
